# Music Streaming UI 🎵

A Spotify-inspired music streaming user interface built using HTML5 and CSS3. The project focuses on recreating a modern music streaming platform layout with a clean design, responsive components, and an intuitive user experience.

## Features

* Modern dark-themed user interface
* Sidebar navigation menu
* Music library and playlist sections
* Interactive music player layout
* Responsive design for different screen sizes
* Built using Flexbox and CSS Grid

## Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
